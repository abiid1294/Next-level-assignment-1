# Next-level-assignment-1
1)  TypeScript এ interface এবং type দুটিই কোনো object এর structure define করতে ব্যবহৃত হয়। interface শুধুমাত্র object এর shape define করার জন্য ব্যবহার হয় এবং এটিকে সহজে extend বা merge করা যায়। type object ছাড়াও union types, primitive types, এবং function signatures define করতে পারে এবং সাধারণত এটিকে পরে extend করা যায় না। তাই object structure এর ক্ষেত্রে interface বেশি flexible, আর বিভিন্নভাবে type define করার জন্য type বেশি versatile এবং powerful.
